# Resume Notes for KinetiKit

## Short project description

KinetiKit is a Python package and command-line toolkit for enzyme kinetics and general chemistry rate-law analysis from generic CSV files.

## Resume bullet options

- Built **KinetiKit**, a tested Python package and CLI for enzyme kinetics and general chemistry rate-law analysis, supporting Michaelis-Menten, substrate inhibition, integrated rate laws, method-of-initial-rates order estimation, CSV ingestion, JSON export, and plot generation.
- Used **agent-assisted software development** to design, implement, test, and document a scientific computing toolkit, applying test-driven development and iterative verification across package APIs, command-line workflows, and example datasets.
- Developed reusable scientific analysis workflows in Python with NumPy, SciPy, pandas, matplotlib, pytest, and packaging standards via `pyproject.toml`.

- General chemistry kinetics

## Interview talking points

- Why generic initial-rate CSVs are more broadly useful than plate-reader-specific raw formats.
- How Michaelis-Menten differs from substrate inhibition and why high-substrate dropoff matters.
- How integrated rate laws can identify zero-, first-, or second-order reactions.
- How method-of-initial-rates data estimates reaction orders through log-linear regression.
- How agentic AI helped accelerate scaffolding, testing, documentation, and verification while still requiring scientific judgment.
